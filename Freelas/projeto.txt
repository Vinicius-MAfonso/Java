Definição do projeto freeLas

Descrição: Aplicação para o gerenciamento de projetos

Objetivo: Resolver a questão de organização de um ou vários projetos

Entidades: 

* Projeto;
    - Nome
    - Contratante
    - Valor por hora
    - Descrição
    - Status
    - Data de criação
    - Data de atualização

Resquisitos:
* Permitir o cadastro de projeto;
* Permitir alterar o projeto;
* Permitir deleter o projeto

Regras de negócio:

* Não irá conter um sistema de login
* Não haverá usuários
* Todo projeto deve pertencer a uma lista

Tecnologias utilizadas:

* Java
* MySQL
